# Fokus ⏱️

Projeto desenvolvido durante o curso **"JavaScript: manipulando elementos no DOM"** da Alura.

O **Fokus** é uma aplicação de temporizador inspirada na técnica **Pomodoro**, criada para ajudar na organização do tempo e no aumento da produtividade. A aplicação permite alternar entre períodos de foco, descanso curto e descanso longo, atualizando dinamicamente elementos da interface.

## 🚀 Funcionalidades

* Temporizador estilo **Pomodoro**
* Alternância entre:

  * Modo foco
  * Descanso curto
  * Descanso longo
* Alteração dinâmica de:

  * Textos
  * Imagens
  * Estilos CSS
  * Atributos e classes
* Interação com a pessoa usuária através de eventos
* Manipulação do **DOM com JavaScript**

## 🧠 Conceitos praticados

* Seleção e manipulação de elementos no **DOM**
* Alteração dinâmica de **atributos e classes**
* Manipulação de **conteúdo HTML e texto**
* Criação e gerenciamento de **eventos**
* Uso de **objetos utilitários do JavaScript**
* Organização de lógica para atualização da interface

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📚 Curso

Projeto desenvolvido como parte do curso da Alura:

**JavaScript: manipulando elementos no DOM**

## 🎯 Objetivo

Praticar conceitos fundamentais de **JavaScript no front-end**, especialmente a manipulação de elementos da página e a criação de interações dinâmicas com a pessoa usuária.
